O que é o Social IFPI?

O Social IFPI é uma plataforma de blog interativa, desenvolvida como um projeto prático para demonstrar a construção de uma aplicação web moderna e funcional. O objetivo é criar um espaço dinâmico onde os usuários possam compartilhar e discutir ideias sobre tecnologia e inovação.

Este projeto simula uma rede social focada em conteúdo, permitindo que os visitantes visualizem as postagens e interajam com elas de diversas formas.

Funcionalidades

  • Criação de Postagens
  • Visualização e Filtragem
  • Edição e Exclusão de Posts
  • Interação com Curtidas
  • Seção de Comentários Recolhível

Tecnologias

  • HTML5
  • CSS3 com Variáveis e Flexbox/Grid
  • TypeScript para um JavaScript robusto
  • Comunicação com API REST
